ĐỀ TÀI: ỨNG DỤNG TRA CỨU THỜI TIẾT (C# WinForms + API)

================================================================================
MỤC TIÊU
================================================================================

Xây dựng một ứng dụng desktop bằng C# WinForms cho phép người dùng tra cứu thời
tiết của các thành phố thông qua API thời tiết. Ứng dụng hiển thị thông tin
thời tiết và lưu lịch sử tìm kiếm.

================================================================================
1. CÔNG NGHỆ SỬ DỤNG
================================================================================

- Ngôn ngữ: C#
- Framework: .NET WinForms
- API: OpenWeather API
- Xử lý dữ liệu: JSON
- Thư viện: HttpClient, Newtonsoft.Json (nếu cần)
- Lưu dữ liệu: File txt

================================================================================
2. CHỨC NĂNG CHÍNH
================================================================================

2.1 Tìm kiếm thời tiết
────────────────────────────────────────────────────────────────────────────────

Người dùng nhập tên thành phố vào ô nhập liệu và nhấn nút "Search".
Chương trình sẽ gọi API thời tiết và hiển thị thông tin thời tiết của thành phố đó.

Thông tin hiển thị gồm:
- Tên thành phố
- Nhiệt độ (°C)
- Độ ẩm (%)
- Tốc độ gió
- Trạng thái thời tiết (Clear, Rain, Clouds, v.v.)
- Icon thời tiết

2.2 Hiển thị icon thời tiết
────────────────────────────────────────────────────────────────────────────────

Ứng dụng hiển thị biểu tượng thời tiết tương ứng với trạng thái thời tiết.

Ví dụ:
- Sunny -> icon mặt trời
- Rain -> icon mưa
- Cloudy -> icon nhiều mây

Icon hiển thị bằng PictureBox trong WinForms.

2.3 Lưu lịch sử tìm kiếm
────────────────────────────────────────────────────────────────────────────────

Mỗi lần người dùng tìm kiếm thành phố, tên thành phố sẽ được lưu vào danh sách lịch sử.
Danh sách lịch sử được hiển thị trong ListBox.

Ví dụ:
Hanoi
Tokyo
Paris
London

Dữ liệu lịch sử được lưu vào file history.txt.

2.4 Xem lại lịch sử
────────────────────────────────────────────────────────────────────────────────

Người dùng có thể click vào một thành phố trong ListBox để tìm lại thời tiết
của thành phố đó.

2.5 Xóa lịch sử
────────────────────────────────────────────────────────────────────────────────

Có nút "Clear History".

Khi nhấn nút này:
- Danh sách lịch sử trong ListBox sẽ được xóa
- File history.txt cũng được cập nhật lại

================================================================================
3. THIẾT KẾ GIAO DIỆN (UI)
================================================================================

Form chính gồm các thành phần sau:

1. Label tiêu đề
   Text: "ỨNG DỤNG TRA CỨU THỜI TIẾT"

2. TextBox nhập thành phố
   Name: txtCity

3. Button tìm kiếm
   Name: btnSearch
   Text: Search

4. Label hiển thị thông tin
   - lblTemperature
   - lblHumidity
   - lblWind
   - lblWeatherStatus

5. PictureBox hiển thị icon thời tiết
   Name: picWeather

6. ListBox hiển thị lịch sử tìm kiếm
   Name: lstHistory

7. Button xóa lịch sử
   Name: btnClearHistory
   Text: Clear History

================================================================================
4. LUỒNG HOẠT ĐỘNG CHƯƠNG TRÌNH
================================================================================

Bước 1:
Người dùng nhập tên thành phố vào TextBox.

Bước 2:
Nhấn nút Search.

Bước 3:
Chương trình gửi request đến API thời tiết.

Bước 4:
API trả dữ liệu JSON.

Bước 5:
Chương trình đọc dữ liệu JSON và hiển thị:
- Nhiệt độ
- Độ ẩm
- Gió
- Trạng thái thời tiết
- Icon

Bước 6:
Tên thành phố được thêm vào lịch sử tìm kiếm.
